dfs深度优先搜索

1. 矩阵中的路径 请设计一个函数，用来判断在一个矩阵中是否存在一条包含某字符串所有字符的路径。路径可以从矩阵中的任意一格开始，每一步可以在矩阵中向左、右、上、下移动一格。如果一条路径经过了矩阵的某一格，那么该路径不能再次进入该格子。例如，在下面的3×4的矩阵中包含一条字符串“bfce”的路径（路径中的字母用加粗标出）

[[“a”,"b”,“c”,“e”]

[“s”,"f”,"c”,“s”]

[“a”,“d”,“e”,"e"]]

代码如下：

class Solution(object):
    def exist(self, board, word):
        """
        :type board: List[List[str]]
        :type word: str
        :rtype: bool
        """
        if not board or not board[0]:
            return False
        m = len(board)
        n = len(board[0])
        marked = [[0]*n for i in range(m)]
        direction = [(0,1),(0,-1),(1,0),(-1,0)]
        
        def dfs(i,j,board,marked,word):
            if not word:return True
            for d in direction:
                tmpx, tmpy = i+d[0], j+d[1]
                if 0<=tmpx<m and 0<=tmpy<n and not marked[tmpx][tmpy] and board[tmpx][tmpy]==word[0]:
                    marked[tmpx][tmpy] = 1
                    if dfs(tmpx,tmpy,board,marked,word[1:]):
                        return True
                    else:
                        marked[tmpx][tmpy] = 0
            return False
        
        for i in range(m):
            for j in range(n):
                if board[i][j] == word[0]:
                    marked[i][j] = 1
                    if dfs(i,j,board,marked,word[1:]):
                        return True
                    else:
                        marked[i][j] = 0
        return False

代码中有几个矩阵搜索中的关键点：

  1. 使用方向数组：
direction = [(0,1),(0,-1),(1,0),(-1,0)]
  1. 使用marked矩阵进行搜索记录:
	marked = [[0]*n for i in range(m)]
  1. 注意每次dfs搜索结束后如果不符合题中条件,要还原marked的值:
    if dfs(tmpx,tmpy,board,marked,word[1:]):
        return True
    else:
        marked[tmpx][tmpy] = 0
